# Mutation-Finder
Compared two DNA sequences and find where mutations happened .

**Project Objective**
--------------------

To compare a reference DNA sequence with a mutated DNA sequence and:

Identify mutation positions

Display the original and mutated nucleotides

Count total mutations

This mimics a basic step used in genetic variation and disease mutation analysis.
